# translate

## 介绍
国际化，网页自动翻译，同谷歌浏览器自动翻译的效果，适用于网站

## 托管
托管于 http://res.zvo.cn/translate/

## 使用
直接在要显示的位置引入js即可

````
<script src="http://res.zvo.cn/translate/translate.js"></script>
````

## 美化 切换语言 的位置
可使用css定义 #translate{} 来定义切换语言的显示。如：

````
<style>
#translate{
	position: absolute;
	top:10px;
	right:10px;
}
</style>
````
